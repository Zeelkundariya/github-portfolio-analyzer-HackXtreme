## ðŸ† The Engineering Signal Algorithm (v10.1)
The engine uses **Heuristic Content Analysis** and **Pattern-Based Activity Scanning** to calculate your grade.

### 1. Architectural Depth (Capped @ 75%)
A deterministic formula measuring high-impact technical signals:
- **Baseline (35 pts)**: Automatic detection of repository structure.
- **Complexity (+5 pts)**: Large-scale repositories (>3MB).
- **Influence (+8 pts)**: Star-to-Fork impact ratio.

- **Security Audit**: Automated scan for exposed secrets, `.env` files, or high-risk credential patterns.
- **Profile Branding**: Bio, social counts, and professional link connectivity.
